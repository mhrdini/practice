# Practice

A repo dedicated to practicing skills and concepts with:

- [React](https://react.dev/) with [Next.js](https://nextjs.org/) at [`/apps/react`](/apps/react/README.md)
- [Vue.js](https://vuejs.org/) with [Nuxt](https://nuxt.com/) at [`apps/vue`](/apps/vue/README.md)

## Quick Start

```bash
pnpm dev
```

This starts the microfrontends (using Turborepo) in a local development environment:

- Next.js at `localhost:3000`
- Nuxt at `localhost:3001`
